## Overview
This Java application demonstrates the implementation of multiple software design patterns to solve common object-oriented problems with reusable, maintainable code.

- Modular architecture with emphasis on clean design

- UML  

## How to Run
1. Clone this repository.  
2. Open the project in an IDE (e.g., IntelliJ, Eclipse).  
3. Build and run `Main.java` to execute the demonstration patterns.
